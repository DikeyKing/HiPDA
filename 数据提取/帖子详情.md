# 帖子详情
## 帖子信息
```
// 有以下三种可能的链接
// 1. https://www.hi-pda.com/forum/viewthread.php?tid=2094735&extra=page%3D1&page=1
// 2. https://www.hi-pda.com/forum/viewthread.php?tid=2094735&rpid=41821617&ordertype=0&page=1#pid41821617
// 3. https://www.hi-pda.com/forum/viewthread.php?tid=2097096&extra=page%3D1
// 若为链接1: 则tid = result[1], page = result[2], pid = nil;
// 若为链接2: 则tid = result[3], page = result[4], pid = result[5];
// 若为链接3: 则tid = result[6], page = 1, pid = nil;
https:\/\/www\.hi-pda\.com\/forum\/viewthread\.php\?tid=(\d+)&extra=page%3D1&page=(\d+)|https:\/\/www\.hi-pda\.com\/forum\/viewthread\.php\?tid=(\d+)&rpid=41821617&ordertype=0&page=(\d+)#pid(\d+)|https:\/\/www\.hi-pda\.com\/forum\/viewthread\.php\?tid=(\d+)&extra=page%3D1
```

```
// result.1为帖子标题

```
